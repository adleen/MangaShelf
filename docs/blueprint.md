# **App Name**: MangaShelf

## Core Features:

- Add Manga Entry: Allows users to add new manga series with details such as title, author, and total volumes.
- Track Volume Ownership: Users can specify which volumes of a manga series they own.
- Mark as Read/Unread: Functionality to mark individual volumes as read or unread to keep track of progress.
- Browse Collection: Display the manga collection in an easy-to-navigate format, with options for sorting and filtering.
- Wishlist Management: Allows users to add manga series or volumes to a wishlist.
- Search by Title: Enables users to quickly find manga within their collection or wishlist by searching for titles.

## Style Guidelines:

- Primary color: Desaturated purple (#B19CD9) for a subtle, calming effect.
- Background color: Light gray (#E0E0E0) for a clean, unobtrusive backdrop.
- Accent color: Dark purple (#4B0082) to highlight interactive elements and important information.
- Body and headline font: 'PT Sans' for a modern and readable user experience.
- Use simple, consistent icons to represent different actions and statuses (e.g., read, unread, owned).
- Emphasize a clean and organized layout that allows for easy browsing and management of the manga collection.
- Implement subtle animations for actions like marking a volume as read or adding it to the wishlist.
